## API Reference

#### Get all items

```http
  GET /api/items
```

| Parameter| Type    | Description                                |
| :--------| :-------| :------------------------------------------|
| `api_key`| `string`| **Required**.ee16cdd2901442cdabd52206240703|

#### Get item

```http
  GET /api/items/${id}
```

| Parameter | Type     | Description                       |
| :-------- | :------- | :-------------------------------- |
| `id`      | `string` | **Required**. Id of item to fetch |
